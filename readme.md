## CLI

### Install 📦

```sh
cargo install moni
```

Or use a binary from
[releases](https://github.com/UltiRequiem/moni/releases/latest) 🤖

### Usage 🏏

```sh
moni
```

This will delete a common list of unwanted directories.

> [List](./src/lib.rs#L6)

Built in help:

```sh
moni --help
```
